Invisible to every
screen capture tool.

MockMate uses OS-level APIs to completely blank the overlay from all recording and sharing software — not just browser capture.

Windows — WDA_EXCLUDEFROMCAPTURE
macOS — NSWindow.sharingType = .none
Zoom, Teams, Meet recordings
OBS, screen recorders, getDisplayMedia

Setup

Running in minutes.

  1. Clone the repo

    Requires Node.js 18+. Works on Windows, macOS, and Linux.

  2. Add your API keys

    Copy .env.example → .env and add your keys. All services have a free tier.

  3. Launch

    MockMate opens as a floating overlay in the top-right of your screen — always on top.

  4. Start your interview

    Open Zoom or Teams. MockMate listens. Answers appear automatically.

# 1. Clone
git clone github.com/vsv2014/MockMate
cd MockMate

# 2. Install
npm install

# 3. Add keys
cp .env.example .env
# edit .env with your API keys

# 4. Launch overlay
npm run dev
